# Invasion-causes-consistent-shifts-towards-acquisitive-plant-communities-but-idiosyncratic-changes-
This repository contains data file and code used for analysis in the manuscript - Invasion causes consistent shifts towards acquisitive plant communities, but idiosyncratic changes in phylogenetic diversity in tropical savannas.
It includes raw data from all analyses presented in the manuscript and is provided to support the findings and ensure transparency for replication studies.

Species_traits: This file contains the raw data used to analyse species-level differences between invasive and native species across multiple traits in two savanna ecosystems. Alongside the species trait data, it also includes,  details on the data sources and whether each dataset was used solely for species-level comparisons or additionally for community-weighted analyses (Analysis type).


Community_weighted_trait: This file contains the raw data used to analyse community-weighted trait mean for multiple traits across invasion gradient in two savanna ecosystems. It contains information on weather the community weighted trait value is of native species in the community (invader included - no), or the entire community (invader included- yes).

Phylogenetic_diversity: This file contains the raw data used to analyse faith's phylogenetic diveristy (fpd) and Net relatedness index phylogenetic diversity (NRI) across invasion gradient in two savannas. It contains information on weather the community weighted trait value is of native species in the community (invader - no), or the entire community (invader - yes).

Secondary_data_reference: Reference of the source from which the secondary trait data was collected.
